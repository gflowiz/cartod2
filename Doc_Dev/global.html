bilink2(dendos) → {Array}

Ajoute à chaque newLeavesObjects des dendo contenus dans le tableau dendos, un attribut incomingFlows contenant l'origine et la destination (lui même ) de chaque flux dont il est la destination.Donc l'attribut incomingFlows est un tableau de longueur n (n étant le nombre de flux reçus par par la feuille en question), est donc chaque élément est un tableau de 2 objets (origine, destination). Cet attribut est utile pour tracer les courbes entre feuillesvia la méthode de edge bundling
